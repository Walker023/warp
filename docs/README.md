## 💡 使用示例

### 在代码中使用翻译
```rust
use crate::i18n::t;

// 简单文本
let text = t!("menu.file");

// 带参数
let msg = t!("messages.welcome", name = "Alice");

// 在 UI 中
Menu::new(t!("menu.file"), options)
```

### 切换语言
```rust
use crate::i18n::switch_locale;

switch_locale("zh-CN");  // 中文
switch_locale("en");     // 英文
```

## 🔄 工作流程

### 添加新翻译
1. 在 `app/locales/en.yml` 添加英文
2. 在 `app/locales/zh-CN.yml` 添加中文
3. 代码中使用 `t!("your.key")`
4. 测试两种语言
5. 提交代码
